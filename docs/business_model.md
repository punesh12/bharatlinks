# Business & Revenue Model

## Revenue Streams

1.  **SaaS Subscriptions (Recurring)**: Monthly and Yearly plans for power users and businesses.
2.  **Enterprise/API Usage (Volume-based)**: Charging based on API call volume for large SMS campaigns (e.g., banks, logistics companies).
3.  **Custom Domain Sales (One-off/Recurring)**: Reselling domains or charging a setup fee for custom branded domains.

---

## Pricing Strategy (INR)

The goal is to be "No-Brainer Affordable" for Indians while maintaining healthy margins.

### 1. Free "Forever" Plan
*   **Target**: Students, Casual sharers.
*   **Price**: **₹0**
*   **Limits**: 
    *   50 Short links / month.
    *   Basic Analytics (No. of clicks only).
    *   Ads on the redirect page (Interstitial) - *Optional Monetization*.

### 2. Starter "Solopreneur" Plan
*   **Target**: Freelancers, Small Instagram shops.
*   **Price**: **₹199 / month** (or ₹1999 / year)
*   **Features**:
    *   500 Short links / month.
    *   **1 Custom Domain** (link.mybrand.com).
    *   QR Code customization (Logo, Color).
    *   Last 30 days analytics.
    *   No Ads.

### 3. Pro "Growth" Plan
*   **Target**: D2C Brands, Agencies, Startups.
*   **Price**: **₹999 / month** (or ₹9999 / year)
*   **Features**:
    *   Unlimited Short links.
    *   **5 Custom Domains**.
    *   **Deep Linking** (App opening).
    *   **WhatsApp Preview Customization**.
    *   Advanced Analytics (City, Device, OS, Time of day).
    *   Team Members (Up to 3).
    *   Export Data (CSV/PDF).

### 4. Organization / API Plan
*   **Target**: SMS Aggregators, Large Corps.
*   **Price**: **Custom** (Starts at ₹5000/mo)
*   **Features**:
    *   API Access (High Limits).
    *   SSO.
    *   Dedicated Account Manager.
    *   SLA for uptime.

---

## Cost Structure
*   **Server/Infrastructure**: AWS/DigitalOcean (Mumbai region) - Scalable.
*   **Domain Economy**: Managing multiple short domains to prevent blacklists.
*   **Marketing**: Content, SEO, Paid ads (minimal initially).
*   **Payment Gateway**: Razorpay/Stripe fees (~2%).

## Unit Economics Example (Pro Plan)
*   **Revenue**: ₹999
*   **Server Cost per user**: ~₹50
*   **Support Cost**: ~₹100
*   **Marketing CAC**: ~₹300 (Amortized)
*   = **Profit**: ~₹549 per user/month (Healthy ~55% Margin)
