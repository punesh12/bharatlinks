## Feature List

### 1. The Essentials (MVP)
*   **Instant Shortening**: Fast URL shortening with custom aliases (e.g., `lnk.in/diwali-sale`).
*   **QR Code Generator**: Basic QR codes for every link.
*   **Link Management**: Edit destination URL after creation (Dynamic Links).
*   **Basic Analytics**: Total Clicks, Device Type (Mobile vs Desktop), Referrer.
*   **UTM Builder & Templates**: Create and save UTM presets (Source, Medium, Campaign) to auto-tag links for consistent tracking.
*   **Workspaces**: Organize links into separate workspaces (e.g., "Personal", "Client A", "Marketing Team").

### 2. The "India-Specific" Power Features
*   **WhatsApp Smart-Share**: 
    *   Pre-generate the text and thumbnail that appears when the link is shared on WhatsApp.
    *   "Click to Chat" link generator for businesses.
*   **UPI Smart Links**: 
    *   Create a shorter link that triggers the user's UPI app directly for payments.
*   **Deep Linking**:
    *   Smart routing: Opens the YouTube app, Instagram app, or Amazon app directly instead of the browser (improves conversion by 2-3x).
*   **Geo-Targeting (State Level)**:
    *   Redirect users from "Maharashtra" to a Marathi landing page and "Tamil Nadu" to a Tamil page.

### 3. Business & Enterprise Features
*   **Branded Domains**: Connect `links.mybrand.com` instead of generic domains.
*   **Bulk Shortening**: CSV upload support for large SMS campaigns.
*   **Team Collaboration**: Multiple seats for marketing teams.
*   **API Access**: Developer-friendly API for integration with CRMs (Zoho, Leadsquared) and SMS gateways.

### 4. Safety & Compliance
*   **Scam Protection**: Localized AI detection for phishing links (crucial in India).
*   **Data Residency**: Guarantee that analytics data stays within Indian servers (DPDP Act compliance ready).
